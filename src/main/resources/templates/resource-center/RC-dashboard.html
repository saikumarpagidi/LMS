Resource Center Dashboard

Welcome to your resource center dashboard. Use the navigation menu above to manage courses and students.

Quick Navigation Help

You can access the following features from the navigation menu:

  • To view all courses, select Courses → Course Details
  • To assign courses to Users, select Courses → Assign Course
  • To view all registered students, select Users → Registered Students
  • To view all registered faculty, select Users → Registered Faculty
  • To view assigned courses, select Courses → Assigned Courses
